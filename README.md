# Judul proyek 
Tugas Laporan Praktikum 6 Aplikasi Product Catalog dengan Fake Store API
# Cara menjalankan
1. dowload atau clone dari git dengan cara
   git clone https://github.com/username/Tugas-6
2. Buka folder project menggunakan Visual Studio Code.
3. Buka terminal terus masuk ke file
   cd Tugas-6
   setelah masuk selanjutnya
   npm install
   setelah selesai instal selanjutnya
   npm run dev
# Fitur-fitur yang diimplementasikan

## Fitur Utama
- Menampilkan daftar produk (Product List)
- Loading Indicator
- Error Handling
- Filter berdasarkan kategori
- Pencarian produk (Search)
- Detail produk dalam modal (Product Detail Modal)
- Responsive Grid Layout

## Fitur Tambahan
- Sorting harga (Low to High & High to Low)
- Add to Cart (Simulasi Keranjang Belanja)
